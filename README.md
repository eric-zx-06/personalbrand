# Eric Zhang - Personal Portfolio

A clean, modern, and elegant personal portfolio website showcasing my work as an NYU student passionate about Data, AI, and Technology.

## 🌟 Features

- **Minimal Design**: Dark theme with purple accents for a sophisticated look
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Subtle hover effects and scroll-based fade-in animations
- **Fast Loading**: Lightweight vanilla HTML, CSS, and JavaScript
- **Professional Typography**: Clean Inter font for excellent readability

## 📋 Sections

- **Hero**: Introduction with name and tagline
- **About**: Personal background and interests
- **Projects**: Showcase of key projects (placeholder cards)
- **Contact**: Ways to get in touch

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for accessibility
- **CSS3**: Modern styling with gradients, flexbox, and grid
- **JavaScript**: Smooth scrolling and intersection observer animations
- **Google Fonts**: Inter font family for typography

## 🚀 Getting Started

### Prerequisites

A modern web browser (Chrome, Firefox, Safari, Edge)

### Running the Website

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/personalbrand.git
   ```

2. Navigate to the project directory:
   ```bash
   cd personalbrand
   ```

3. Open `index.html` in your web browser

That's it! The website will load instantly as it's a static site with no build process required.
